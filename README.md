# ENCS5337 DV Project - BIRD Verification



## Project Overview



This project implements a complete SystemVerilog verification environment for the BIRD traffic routing module. The verification environment validates both Local and Remote packet routing behavior according to the project specification.



The environment includes a Driver, Monitors, Reference Model, Scoreboard, Functional Coverage, and multiple directed test cases to verify DUT functionality.
